# Matomo GravatarPictures Plugin

## Description

This Matomo Plugin looks for Gravatar Images for your Visitors.

## Matomo Version

min 3.13.0-stable

## Installation

Simply go to the Matomo Marketplace and search for "GravatarPictures"
